setup & troubleshooting

Server URL: use the HTTPS plex.direct address (e.g. https://10-0-0-12.hash.plex.direct:32400) so this HTTPS page isn't blocked by mixed-content rules. Find it via https://plex.tv/api/resources?includeHttps=1&X-Plex-Token=YOURTOKEN — use the connection marked local="1" when the tablet shares Wi-Fi with the server.

Token (on iOS): at app.plex.tv → open any item → ··· → Get Info → View XML, then tap Share → Copy and grab the value after X-Plex-Token=. Stored only in this browser.

Won't play / no visualizer: almost always CORS. Add https://junk.timbornholdt.com under Plex → Settings → Network → List of allowed CORS origins, then reconnect.

Skins & visualizer: right-click the player for Skins, Visualizer → Milkdrop, or Double Size. More skins at the Skin Museum.
