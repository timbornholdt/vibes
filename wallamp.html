setup & troubleshooting

1. Use the HTTPS plex.direct address. This page is served over HTTPS, so the browser will block audio loaded from a plain http://192.168.x.x address (mixed content). Plex gives every server a valid-cert HTTPS hostname that looks like https://10-0-0-12.abc123…def.plex.direct:32400. Find it in Plex by clicking a server → … → Get Info, or in the URL of the web app. Use that here.

2. Get your token. In Plex, play any track → click … → Get Info → View XML. Copy the X-Plex-Token=… value out of the address bar. It's stored only in this browser's localStorage — nothing is sent anywhere except your own server.

3. If audio won't play / no visualizer: that's almost always CORS. Plex normally allows it, but if not, add this page's origin (https://junk.timbornholdt.com) under Settings → Network → List of allowed CORS origins on your Plex server, then reconnect.

4. iPad / Safari or FLAC library: tick force mp3 so Plex transcodes on the fly. Direct play (unticked) is lighter and best for MP3/AAC libraries.

Skins & visualizer: right-click the player for Skins and Visualizer → Milkdrop, or Double Size for a bigger wall display. Grab more skins at the Winamp Skin Museum and paste a .wsz URL below.
